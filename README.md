# 🍔 Swiggy Performance Dashboard

## 📌 Project Overview
This is an end-to-end Data Analytics project built using Excel, MySQL, and Tableau.

The project analyzes Swiggy food delivery data to identify:
- Customer ordering behavior
- Revenue trends
- Cancellation analysis
- Peak ordering hours
- Top-performing restaurants
- Customer retention insights

---

## 🛠 Tools Used
- Excel
- MySQL Workbench
- Tableau Desktop

---

## 🔄 Project Workflow
1. Data Cleaning and Transformation in Excel
2. CSV conversion and database loading in MySQL
3. Data modeling using Primary and Foreign Keys
4. SQL joins and KPI validation
5. View creation for reporting
6. Dashboard creation in Tableau

---

## 📊 KPIs Included
- Total Orders
- Total Revenue
- Average Order Value
- Revenue Per Customer
- Delivery vs Cancellation %
- Peak Time Analysis
- Repeat vs New Customers
- Top Restaurants by Revenue

---

## 📈 Dashboard Insights
- Around 79% of orders were successfully delivered.
- Repeat customers contributed more revenue than new customers.
- Italian and Indian cuisines showed highest customer preference.
- Peak order activity was observed during evening hours.
- Revenue trends fluctuated across different months and locations.
